CanvasJS Chart Annotator

This plugin allows you to annotate CanvasJS charts. Check out Github Repository for more info / examples.

How to Use?

  • Render the chart
  • Call addAnnotations method
var chart = new CanvasJS.Chart("chartContainer", {
    .
    .
    .
    //Chart Options
    .
    .
    .
});
chart.render()
chart.addAnnotations();
